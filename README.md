# 🛡️ LoL Hub

A modern full-stack League of Legends fan wiki built with React, Node.js, Express and MongoDB.

LoL Hub allows players to explore champions, items, summoner spells, runes and regions through a fast, responsive and bilingual interface.

> This project is intended for educational purposes and is not affiliated with or endorsed by Riot Games.

---

# Features

- Champions explorer
- Champion detail pages
- Favorite champions
- Items database
- Summoner Spells
- Rune system
- Regions explorer
- User authentication
- Password recovery via email
- User profile
- Contact form
- English / Spanish support
- Responsive design
- SEO optimized
- Google Search Console ready

# Installation

Clone the repository

```bash
git clone https://github.com/hazajulian/LoL-Hub.git
```

Install frontend

```bash
cd frontend
npm install
npm run dev
```

Install backend

```bash
cd backend
npm install
npm run dev
```
