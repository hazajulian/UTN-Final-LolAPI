# 🛡️ LoL Hub

LoL Hub es una wiki fan-made de League of Legends desarrollada con React, Node.js, Express y MongoDB.

Permite explorar campeones, objetos, runas, hechizos de invocador y regiones mediante una interfaz moderna, rápida y completamente responsive.

> Este proyecto fue desarrollado con fines educativos y no está afiliado ni respaldado por Riot Games.

---

# Características

- Explorador de campeones
- Detalle completo de campeones
- Campeones favoritos
- Base de datos de objetos
- Hechizos de invocador
- Sistema de runas
- Explorador de regiones
- Autenticación de usuarios
- Recuperación de contraseña por correo
- Perfil de usuario

- Español e inglés

- Optimizado para SEO
- Preparado para Google Search Console

# Instalación

Clonar el repositorio

```bash
git clone https://github.com/hazajulian/LoL-Hub.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```
